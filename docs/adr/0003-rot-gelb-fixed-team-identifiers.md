# Rot and Gelb are the canonical team identifiers

Each Spiel has exactly two teams, always identified as Rot and Gelb. These are not configurable names — they are fixed values in the schema and domain language. This reflects how the group has always referred to their teams (coloured bibs), is embedded in the paper match sheet (Eigentor von Rot / Eigentor von Gelb), and keeps the data model simple. A generic "Team A / Team B" or dynamic team naming would add configuration complexity for no practical benefit to this group.
